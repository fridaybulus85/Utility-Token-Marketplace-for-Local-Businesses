# 🏪 Utility Token Marketplace for Local Businesses

A decentralized marketplace where local businesses can issue their own utility tokens (like coffee credits, barber shop hours, or service vouchers) that customers can purchase, trade, and redeem. This helps strengthen local economies by bringing them on-chain! ☕💇‍♂️

## 🌟 Features

- **🏢 Business Registration**: Local businesses can register and create their profiles
- **🪙 Token Creation**: Businesses can issue custom utility tokens for their services
- **💰 Token Purchase**: Customers can buy tokens directly from businesses
- **🔄 P2P Trading**: Users can trade tokens with each other through order books
- **🎫 Token Redemption**: Redeem tokens for actual services at businesses
- **👤 User Profiles**: Track trading history and build reputation scores
- **📊 Marketplace Analytics**: View token prices, trading volumes, and business metrics

## 🚀 Getting Started

### Prerequisites
- Clarinet CLI installed
- Stacks wallet for testing

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd utility-token-marketplace
```

2. Install dependencies:
```bash
clarinet requirements
```

3. Run tests:
```bash
clarinet test
```

4. Deploy locally:
```bash
clarinet integrate
```
